Note: This project was created with Apache NetBeans IDE 9.0

The Main class is actually Input.java, so keep that in mind if you are creating a new project to run this program and are unable to open this project directly. 

This project requires JDK version 11, download it from here: https://www.oracle.com/technetwork/java/javase/downloads/jdk11-downloads-5066655.html

There is also an included jar executable file that you can run, you just will need above JDK 11.
